# Task list company

Краткое описание проекта

# Install
1.
```bash
git clone https://github.com/shreyner/task-list-company.git
```
2.
```bash
cd task-list-company
```
3.
```bash
npm install
```
4.
```bash
npm start
```

# Production Build
1. [Install](#Install) 1 - 3
2.
```bash
npm run build
```
